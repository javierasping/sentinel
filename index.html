Bienvenidos

    Francisco Javier Cruces Doval

    Administrador de sistemas

    Tengo experiencia en administración de sistemas, abarcando sistemas operativos como GNU/Linux y Windows, virtualización, servicios en la nube, y herramientas de gestión. Además, tengo conocimientos en bases de datos, redes y seguridad, lo que me permite abordar una amplia gama de desafíos en el ámbito de la administración de sistemas

    Competencias

    Virtualización

    Experiencia en la virtualización utilizando KVM para la creación y gestión de máquinas virtuales, así como en la administración de contenedores LXC.

    Docker

    Experiencia en la construcción y gestión de contenedores Docker, incluyendo la creación de imágenes, redes y el uso de Docker Compose para definir y ejecutar aplicaciones multi-contenedor.

    Kubernetes

    Experiencia en la configuración y gestión de clústeres Kubernetes, incluyendo Pods, ReplicaSets y Deployments. Manejo de servicios así como almacenamiento.

    Infraestructura como código

    Uso de Ansible y Vagrant para la creación y gestión de infraestructuras como código.

    Cloud

    Trabajo con la mayoría de las principales nubes, como GCP, AWS, Azure, etc.

    OpenStack

    Trabajo con instancias, almacenamiento y redes en OpenStack.

    Linux

    Experiencia con Debian, Ubuntu, RHEL, CentOS y Fedora. Administración de servidores, configuración y mantenimiento de servicios.

    Git

    Experiencia en desarrollo basado en Git. Principalmente, uso GitHub. También, tengo experiencia en trabajar con GitLab.

    Servidores web

    Experiencia en la administración de servidores web Apache y Nginx, incluyendo configuración y mantenimiento .

    Proxy Inverso

    Configuración y gestión de proxys inversos para redirigir solicitudes de usuarios a servidores backend, utilizando soluciones como Nginx y Apache.

    Balanceadores de Carga

    Experiencia en la implementación y configuración de balanceadores de carga como HAProxy para distribuir tráfico de red y mejorar la disponibilidad de servicios.

    DNS

    Experiencia en la configuración y administración de servidores DNS utilizando BIND9. Capaz de gestionar zonas, registros y resolver problemas de resolución de nombres en redes locales y dominios externos.

    Correo Electrónico

    Experiencia en la configuración y administración de servidores de correo electrónico utilizando Postfix. Implementación de filtros de spam con SpamAssassin y el uso de herramientas como Dovecot para la gestión de buzones de correo.

    VPN

    Experiencia en la configuración y administración de VPNs como OpenVPN y WireGuard, así como IPSec con Cisco y StrongSwan.

    Cortafuegos

    Experiencia en la configuración y gestión de cortafuegos utilizando Iptables y Nftables para la gestión de reglas de firewall en sistemas Linux, así como dispositivos de Fortinet.

    Cisco

    Configuración y gestión de routers con IOS, Switching (VLAN, QoS, Port Trunking) y Administración (Mantenimiento, Seguridad, VPN, VLAN, WiFi, IPv6).

    Bases de Datos

    Experiencia en la administración y gestión de diversas bases de datos como Oracle, MySQL, PostgreSQL y MongoDB, así como conocimientos en SQL y PL/SQL.

    CI/CD

    Experiencia en la implementación de pipelines de integración continua y despliegue continuo utilizando GitHub Actions y Jenkins. Pruebas automáticas, validación de contenido y despliegues de aplicaciones.
